# Kenya National Digital Asset Framework

Official portal for Kenya's National Digital Asset Recovery, Security, and VASP Tax Enforcement Framework.

## Setup Instructions

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```

## Project Structure

- `index.html` - Main landing page
- `src/main.js` - JavaScript entry point
- `src/style.css` - Custom styles
- `vite.config.js` - Vite configuration

## Technologies Used

- Vite
- Tailwind CSS (CDN)
- AOS (Animate On Scroll)
- Google Fonts (Inter & Playfair Display)
